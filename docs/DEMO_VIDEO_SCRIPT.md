## üé¨ Scene 2: Basic Note Creation (0:20 - 0:45)

**[Screen: Focus on drum grid]**

**Narrator:**
> "Let's start with the basics. In simple mode, just click any cell to add a note."

**Action:**
1. Click several cells to add notes (Hi-Hat, Snare, Kick)
2. Click existing notes to remove them
3. Show the notes appearing with icons

**Narrator:**
> "Each note shows an icon representing its articulation - normal hits, accents, ghost notes, and more."

**Action:**
- Point to different note icons (normal, accent, ghost)

---

## üé¨ Scene 3: Articulation Selection (0:45 - 1:10)

**[Screen: Right-click context menu]**

**Narrator:**
> "Right-click any cell to choose from different articulations."

**Action:**
1. Right-click a Hi-Hat cell
2. Show the articulation menu with icons
3. Select "Accent" - note changes to accent icon
4. Right-click a Snare cell
5. Select "Ghost Note" - note changes to ghost icon
6. Right-click a Kick cell
7. Select "Heel-Toe" - note changes to heel-toe icon

**Narrator:**
> "Each drum voice has its own set of articulations - from hi-hat techniques like half-open and pedal, to snare techniques like rim shots and drags."

---

## üé¨ Scene 4: Advanced Edit Mode (1:10 - 1:35)

**[Screen: Toggle advanced mode]**

**Narrator:**
> "When you're working with articulations frequently, press 'E' to toggle Advanced Edit Mode."

**Action:**
1. Press `E` key - show toggle switching to ON
2. Left-click a cell - articulation menu opens
3. Select an articulation
4. Left-click another cell - menu opens again

**Narrator:**
> "In Advanced Mode, left-click opens the articulation menu directly, making it faster to work with different techniques."

**Action:**
- Press `E` again to toggle back to simple mode

---

## üé¨ Scene 5: Drag-to-Paint (1:35 - 2:05)

**[Screen: Drag-to-paint demonstration]**

**Narrator:**
> "Here's where it gets powerful. Hold Ctrl and drag to paint notes across multiple cells."

**Action:**
1. Hold `Ctrl` (show cursor changing to crosshair)
2. Drag across Hi-Hat row - notes appear as you drag
3. Drag across Snare row - more notes appear
4. Drag across Kick row

**Narrator:**
> "This is perfect for creating hi-hat patterns or rapid fills. And to erase? Just hold Alt and drag."

**Action:**
1. Hold `Alt` (show cursor changing to not-allowed)
2. Drag across some notes - they disappear
3. Release Alt

---

## üé¨ Scene 6: Bulk Operations (2:05 - 2:40)

**[Screen: Bulk operations dialog]**

**Narrator:**
> "Need to fill an entire measure quickly? Click any voice label to open bulk operations."

**Action:**
1. Click "Hi-Hat" label
2. Show the bulk operations dialog
3. Hover over patterns to show descriptions:
   - "All On" - fills every position
   - "Upbeats" - fills the "&" positions
   - "Eighths" - standard eighth note pattern

**Narrator:**
> "Let's create a classic rock beat. First, hi-hats on eighths..."

**Action:**
1. Click "Eighths" pattern - hi-hats fill in
2. Close dialog

**Narrator:**
> "Then, snare on the backbeat..."

**Action:**
1. Click "Snare" label
2. Click "Backbeat" pattern - snares on 2 and 4
3. Close dialog

**Narrator:**
> "And kick on the downbeats."

**Action:**
1. Click "Kick" label
2. Click "Four on Floor" pattern - kicks on 1, 2, 3, 4
3. Close dialog

**Narrator:**
> "There's your rock beat in seconds!"

---

## üé¨ Scene 7: Playback & Tempo (2:40 - 3:00)

**[Screen: Playback controls]**

**Narrator:**

**Action:**
1. Click Play button
2. Show playhead moving across grid
3. Notes highlight as they play
4. Let it loop once

**Narrator:**
> "Adjust the tempo with the slider, or add swing for a more human feel."

**Action:**
1. Adjust tempo slider (show BPM changing)
2. Adjust swing slider
3. Let it play with swing

## üé¨ Scene 8: Undo/Redo (3:00 - 3:20)

**[Screen: Undo/Redo controls]**

**Narrator:**
> "Made a mistake? No problem. Use Ctrl+Z to undo, or click the undo button."

**Action:**
1. Make some changes (add/remove notes)
2. Press `Ctrl+Z` - changes revert
3. Press `Ctrl+Z` again - more changes revert
4. Press `Ctrl+Shift+Z` - redo the changes
5. Click the Undo/Redo buttons to show UI controls

**Narrator:**
> "Every edit is tracked, so you can experiment freely."

### Visual Effects to Add:

### Audio:
- **Background music:** Subtle, upbeat, non-intrusive
- **Sound effects:** Actual drum sounds from the app
- **Voiceover:** Clear, enthusiastic, professional

### Screen Recording Settings:
- **Resolution:** 1920x1080 (Full HD)
- **Frame rate:** 60fps for smooth playback animation
- **Browser:** Chrome or Firefox (latest version)
- **Window size:** Maximize or set to 1920x1080

### Before Recording Checklist:
- [ ] Clear browser cache
- [ ] Disable browser extensions
- [ ] Hide bookmarks bar
- [ ] Set zoom to 100%
- [ ] Prepare example patterns in advance
- [ ] Test all features work smoothly
- [ ] Have a script/teleprompter ready
